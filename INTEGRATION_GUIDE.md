Load customers from Firebase:\n\n```tsx\n'use client';\n\nimport { useEffect, useState } from 'react';\nimport { MainLayout, CustomerList } from '@/components';\nimport { getCustomers } from '@/lib/firestore';\nimport type { Customer } from '@/types';\n\nexport default function CustomersPage() {\n  const [customers, setCustomers] = useState<Customer[]>([]);\n  const [loading, setLoading] = useState(true);\n\n  useEffect(() => {\n    const loadCustomers = async () => {\n      try {\n        const data = await getCustomers();\n        setCustomers(data);\n      } catch (error) {\n        console.error('Failed to load customers:', error);\n      } finally {\n        setLoading(false);\n      }\n    };\n\n    loadCustomers();\n  }, []);\n\n  return (\n    <MainLayout title=\"Customers\">\n      <CustomerList customers={customers} isLoading={loading} />\n    </MainLayout>\n  );\n}\n```\n\n---\n\n## рҹҺЁ Customization\n\n### Change Colors\n\nEdit `tailwind.config.js`:\n\n```js\ntheme: {\n  extend: {\n    colors: {\n      primary: '#1a3a6b',    // Change primary color\n      accent: '#f97316',     // Change accent color\n    }\n  }\n}\n```\n\n### Customize Component\n\nAdd className:\n\n```tsx\n<Button className=\"rounded-none uppercase\">Custom Button</Button>\n<Card className=\"bg-gradient-to-r from-blue-50 to-indigo-50\">\n```\n\n---\n\n## рҹ“Ӣ Checklist\n\n- [ ] Copy `components/` folder\n- [ ] Copy `lib/utils.ts`\n- [ ] Install any missing dependencies (should already be in package.json)\n- [ ] Update dashboard pages\n- [ ] Update entity pages\n- [ ] Wire up Firebase\n- [ ] Test all pages\n- [ ] Test on mobile\n- [ ] Deploy to Vercel\n\n---\n\n## рҹ“ҡ Reference\n\n- **COMPONENTS.md** -
